## 🚀 Getting Started

### Prerequisites
- Python 3.7+
- Git
- Flask
- Basic Linux command-line knowledge

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/ADVAIT135/Applied-Software-Engineering-Fundamentals-Specialization.git
cd Applied-Software-Engineering-Fundamentals-Specialization
```

2. **Set up a virtual environment:**
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. **Install dependencies:**
```bash
pip install flask
```

### Running Projects

#### Emotion Detector Application
```bash
cd "Developing AI Applications with Python and Flask/EMOTION DETECTOR"
python server.py
```

Visit `http://localhost:5000` in your browser to access the application.
